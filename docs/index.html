"UNLIMITED" REQUEST API

Just use messages and language as params to the API URL below

API for your product :


Example Request: GET https://api.simsimi.net/v1/?text=hi&lang=vi_VN

  • text : Sentence that a user entered in your chatbot(1000 characters max.)
  • data : simsimi.net
  • lang : Language code (vi, en, ph, zh, ch, ru, id, ko, ar, fr, ja, es, de, ...)
  • cf : Chatfuel Json [True] or [False]

Response ChatFuel: {"messages":[{"text":"Xin chào"}],"database":"true"}

Response: {"success":"Xin chào","database":"true"}
